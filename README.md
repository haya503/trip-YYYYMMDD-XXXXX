旅行用テンプレートリポジトリ

  - YYYY

  - MM

  - XXXXX

## コマンド一覧

|     コマンド名     | 内容                                        |
| :----------------: | :------------------------------------------ |
| `npx honkit build` | ローカルのhonkitでbuildを行う               |
| `npx honkit serve` | ローカルのhonkitのサーバー起動              |
|  `npm run deploy`  | githubにgh-pagesブランチに_bookの内容をpush |
